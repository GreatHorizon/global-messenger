# global-messenger
  Мессенджер похожий на телеграм 
## Функционал
  1)Регистрация

  2)Список контактов
  
  3)Передача тектовых сообщений между пользователями
  
## Технологии
  1)Android Studio. Используемый язык: Kotlin
  
  2)Node.js + express.js
  
  3)MySql
  
  4)websocket
